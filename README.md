# Dvd-bouncing-logo-snake-arcade
This is the snake arcade dvd bouncing logo source code and files where you can find here: https://www.daboiyoshi.com/home/screen-saver.
# how to use
- download as a `.zip`.
- unzip it.
- double click the `index.html` file.
- And then the file will run in your browser and it should have the snake arcade logo bouncing around your screen.
- tip: it works very well on 4:3 screen ratio.
